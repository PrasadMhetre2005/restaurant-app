## 🎨 Customization (10 Minutes)

### Change Restaurant Data
Edit `src/data.js`:
```javascript
const restaurantData = {
    name: "Your Restaurant Name",
    cuisine: "Your Cuisine",
    location: "Your City",
    rating: 4.8,
    // ... update menu items, reviews, offers
};
```

### Update Menu Items
In `src/data.js`, modify the `menu` array:
```javascript
{
    id: 1,
    name: "Your Dish Name",
    price: 100,
    rating: 4.5,
    category: "main",
    description: "Description",
    image: "https://image-url.jpg"
}
```

### Change Colors
Edit `src/style.css`:
```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
}
```

### Update Images
Replace image URLs in `src/data.js` with your images (Unsplash, Pexels, or your own)
